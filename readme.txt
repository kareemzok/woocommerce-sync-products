Upload `sync-for-woocommerce` folder  to the `/wp-content/plugins/` directory.2. Activate the plugin through the 'Plugins' menu in WordPress.3. From left wordpress menu click on 'Woo Commerce Sync' then plugin settings4. From the setting you can control this plugin.	== Frequently Asked Questions === Does Woo commerce sync supports shortcodes? =* No no need as its a sync plugin.= Which WordPress versions are supported? =* To use all features, a minimum of version WordPress 3.0 with PHP 5.5 is required. * Earlier versions are also supported.= Does Woo commerce sync supports a demo? =* Yes it does.= Does Woo commerce sync supports multilanguage? =* Not not yet.= Is Woo commerce sync free? =* Currently the plugin is free.== Screenshots ==1. == Upgrade Notice ==Just the usual, deactivate plugin, replace files, activate.If you directly updated your plugin from plugin page and changes didn't work please deactivate it then activate.== Changelog === 1.0 =* First Beta Release
